# Commento Translation Protocol

PicoDB may produce an original English translation of Pico's Italian *Commento* from the controlled Biblioteca Italiana TEI witness. The translation is an original project artifact and must not be reconstructed from Jayne or Stanley.

## Workflow

1. Translate from the Italian TEI witness.
2. Record section coverage in `artifacts/translations/commento/`.
3. Keep technical vocabulary stable through the glossary.
4. Add translator's notes for source names, doctrinal stakes, and philological uncertainty.
5. Compare Stanley and Jayne only as translation/reception witnesses.
6. Link each translated section to the exhaustive section-summary workflow.

## Citation Rule

When using the translation in essays, cite the Italian TEI source and the PicoDB translation artifact. If a reading depends on Stanley or Jayne, cite that witness separately as a translation comparison.
