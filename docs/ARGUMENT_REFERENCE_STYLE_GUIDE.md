# Argument and Reference Style Guide

## Purpose

PicoDB must not become a stack of pleasant summaries. It needs argument maps and reference registers that preserve how each text reasons, what it cites, what it assumes, and where it enters Pico studies.

## Argument Mapping

For every section, identify:

- Main thesis: the strongest claim the section advances.
- Problem: the question, contradiction, confusion, gap, or inherited interpretation that motivates the section.
- Stakes: why the section matters for Pico, Renaissance studies, magic studies, Kabbalah, astrology, theology, or textual history.
- Method: philology, close reading, doctrinal analysis, historical reconstruction, source study, comparison, edition criticism, reception history, or synthesis.
- Premises: explicit assumptions or authorities.
- Moves: the ordered steps by which the section proceeds.
- Evidence: examples, texts, citations, dates, manuscripts, editions, places, persons, or conceptual distinctions.
- Conclusion: what the section establishes or leaves open.
- Consequence: what future summaries, ontology terms, website pages, biographies, or gap records must update.

## Argument Move Vocabulary

Use this controlled vocabulary when marking moves:

- `defines`
- `distinguishes`
- `rejects`
- `qualifies`
- `reconciles`
- `hierarchizes`
- `allegorizes`
- `historicizes`
- `textualizes`
- `philologizes`
- `transmits`
- `contests`
- `appropriates`
- `warns`
- `opens_problem`

## Reference Register

Create a reference register for each section when references are dense.

Minimum fields:

| Reference | Type | Tradition/Field | Local Role | Evidence | Status |
|---|---|---|---|---|---|

Types:

- person
- work
- biblical_text
- classical_text
- doctrine
- manuscript
- edition
- place
- institution
- event
- language
- term
- scholar

Statuses:

- `identified`
- `needs_review`
- `uncertain_ocr`
- `variant_name`
- `requires_external_check`

## Coverage Warnings

Mark a summary as incomplete if any of these are true:

- The section contains named references not listed in the register.
- The section includes a technical distinction not explained.
- The summary records conclusions but not the steps of the argument.
- The summary does not say how evidence supports the claim.
- A scholarly section is summarized without naming the prior interpretation it revises.
- A primary text is summarized without its scriptural, classical, or philosophical references.
- A late Pico text is summarized without textual-transmission status.

## Pico-Specific Reference Priorities

Always track references in these zones:

- Biblical: Genesis, Psalms, Moses, Paul, John, Matthew, Job, Jacob, Exodus, Ezekiel.
- Patristic and medieval Christian: Augustine, Dionysius, Aquinas, Albert, scholastic commentators.
- Classical and late antique: Plato, Aristotle, Plotinus, Proclus, Porphyry, Iamblichus, Macrobius, Cicero, Plutarch.
- Jewish and Kabbalistic: Hebrew, Chaldean/Aramaic, Targum, Zohar, Maimonides, Nahmanides, Gersonides, Recanati, Mithridates, del Medigo.
- Esoteric/philosophical: Hermetica, Asclepius, Orphic Hymns, Chaldean Oracles, Pythagorean materials, prisci theologi.
- Renaissance network: Ficino, Poliziano, Lorenzo de' Medici, Benivieni, Savonarola, Gianfrancesco Pico.
- Modern scholarship: Copenhaver, Farmer, Wirszubski, Kristeller, Howlett, Akopyan, Allen, Edelheit, Dougherty, Crofton Black.
